הסקת טיפוס

הסקת טיפוס פירושה שאם לא תגיד למהדר את הסוג ואם הוא יכול להחליט בעצמו, הוא יחליט. המהדר תמיד צריך לדעת את סוג המשתנים, אך לא תמיד צריך להגדיר לו זאת. למעשה, בדרך כלל אינך צריך לעשות זאת. לדוגמא, עבור my_number = 8 , let my_number יהיה i32 . הסיבה לכך היא שהמהדר בוחר i32 למספרים שלמים אם אינך אומר זאת. אבל אם אתה אומר תן למספר שלי: u8 = 8 , זה יהפוך את המספר שלי ל- u8 , כי אמרת את זה u8 .

אז בדרך כלל המהדר יכול לנחש. אבל לפעמים אתה צריך לספר את זה משתי סיבות:

אתה עושה משהו מאוד מורכב והמהדר לא יודע את הסוג שאתה רוצה.
אתה רוצה סוג אחר (למשל, אתה רוצה i128, ולא i32).

כדי לציין סוג, הוסף נקודתיים אחרי שם המשתנה.

fn main() {
    let small_number: u8 = 10;
}

עבור מספרים, אתה יכול לומר את הסוג שאחרי המספר. אתה לא צריך רווח - פשוט הקלד אותו מיד אחרי המספר.

fn main() {
    let small_number = 10u8; // 10u8 = 10 of type u8
}

אתה יכול גם להוסיף _ אם אתה רוצה להקל על המספר לקריאה.

fn main() {
    let small_number = 10_u8; // This is easier to read
    let big_number = 100_000_000_i32; // 100 million is easy to read with _
}

ה- _ אינו משנה את המספר. זה רק כדי להקל עליך לקרוא. וזה לא משנה בכמה _ אתה משתמש:

fn main() {
    let number = 0________u8;
    let number2 = 1___6______2____4______i32;
    println!("{}, {}", number, number2);
}
